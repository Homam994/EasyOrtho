⚡ كيفية إعداد JSONBin.io (مجاني تماماً)

  1. افتح jsonbin.io وأنشئ حساباً مجانياً
  2. من لوحة التحكم اضغط "+ Create Bin" وضع أي محتوى مثل {} ثم احفظ
  3. انسخ BIN ID من عنوان URL أو من صفحة الـ Bin (سلسلة مثل 64a1b2c3d4e5f6...)
  4. من القائمة الجانبية اذهب إلى API Keys ثم أنشئ مفتاحاً جديداً وانسخه
  5. أدخل المعلومات أدناه واضغط اختبار وحفظ

⚠️ هذه المعلومات تُحفظ في المتصفح فقط — يجب إدخالها في كل متصفح/جهاز تدير منه الموقع.
